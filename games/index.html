Scrunch
A modded Scratch editor coded in JavaScript

Scratch is very limiting, and it only runs on the website. Scrunch is a extremely powerful version of Scratch allowing for powerful new functions, like communicating with external devices and having controller support. It lets you change the screen ratio, load new extensions and runs projects faster (But not as fast as TurboWarp... :( ) than Scratch can. The built in TubbyWhop packager also lets you convert your projects to Python, C# and HTML, CSS and Javascript.
